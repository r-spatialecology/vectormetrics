# Example map (random cluster neutral landscape model).

An example map to show landscapetools functionality generated with the
\`nlm_randomcluster()\` algorithm and coerced to vector data
(multipolygon).

## Usage

``` r
vector_landscape
```

## Format

A sf object.

## Source

Simulated neutral landscape model with R.
https://github.com/ropensci/NLMR/
